Um utilitário de desktop desenvolvido em Python para medir e monitorar a qualidade da sua conexão de internet. O projeto combina uma interface amigável com a geração de logs e gráficos de desempenho em tempo real.

📋 Sobre o Projeto
Este projeto nasceu da necessidade de monitorar oscilações de rede de forma simples e visual. Ele utiliza a infraestrutura do Speedtest.net para obter dados precisos de Download, Upload e Ping, salvando tudo em um histórico local para consultas futuras.

🛠️ Tecnologias Utilizadas
Python 3.x: Linguagem base.

Tkinter: Para a interface gráfica (GUI).

Speedtest-cli: Motor de testes de rede.

Matplotlib: Geração do gráfico de desempenho da sessão.

Datetime: Registro de carimbo de data/hora nos logs.

✨ Funcionalidades
Teste em Tempo Real: Medição de Download, Upload e Latência com um clique.

Logs Automáticos: Gravação de cada teste no arquivo historico_internet.txt.

Visualização Gráfica: Gráfico evolutivo da velocidade de download durante a sessão ativa.

Interface Dark/Clean: Visual focado na clareza das informações.
